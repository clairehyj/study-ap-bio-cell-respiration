Test Page - AP Bio Quiz with TXT

To test the application:

  1. Open index.html in your browser
  2. Click "Start Quiz" button
  3. Answer the questions by clicking on options
  4. Click "Submit Answer" after selecting
  5. Read TXT members' reactions and explanations
  6. Click "Next Question" to continue
  7. View your final score at the end

Features Implemented:

  • ✅ TXT K-POP themed design with member avatars
  • ✅ Chat bubble interface for questions and responses
  • ✅ Questions parsed from questions.md file
  • ✅ Interactive quiz with score tracking
  • ✅ Encouraging messages from TXT members
  • ✅ Responsive design for mobile devices
  • ✅ Results modal with percentage score
  • ✅ Teenage-friendly aesthetic with rounded corners
  • ✅ Gradient colors and animations
  • ✅ Progress bar showing quiz completion

GitHub Pages Deployment:

To deploy on GitHub Pages:

  1. Push this repository to GitHub
  2. Go to Settings → Pages
  3. Select source as "Deploy from a branch"
  4. Choose "main" branch and "/ (root)" folder
  5. Save and wait for deployment
  6. Access at: https://[username].github.io/[repo-name]/
